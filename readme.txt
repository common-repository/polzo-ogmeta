== Description ==

Plugin insert open graph metadata on your WordPress site. The plugin still in beta, so please send bug reports to me by <a href="mailto:akurganow@polzo.ru?subject=ogMeta">e-mail</a>. If you want to help with the translation, write to the same e-mail, but with the notation "Translation [language]"

* More information: <a href="http://polzo.ru/ogmeta">Polzo.ru</a>
* See: <a href="http://ogp.me/">OPG.me</a>

== Installation ==

* Upload the plugin folder to your blog
* Activate it.
